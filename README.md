# Enery Utility Platform

## HOW TO RUN THE APP

1.  clone this repo
2. run in [/energy]: docker build -t energy .
2. run in [/energy]: docker compose up -d
3. run in [/energy-fe]: docker build -t vue .
4. run in [/energy-fe]: docker compose up -d
